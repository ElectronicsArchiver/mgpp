## Features

- Manage the Dependencies
- Download the Game
- Debug your Mod
- Work with Other Mods
- Separate Working Space
- Manage your Assets
- Generate Resource Class
- Deploy on Android

Please check the [homepage](https://plumygame.github.io/mgpp/) to obtain more information.
