## Notations
The notation always represents a term or indicates a noun.
### Latest notation
Commonly used to represent the latest version.
=== "Groovy"

    ```groovy
    latest
    ```

=== "Kotlin"

    ```kotlin
    latest
    ```
### Local properties notation
Commonly used to represent the `local.properties` file.
=== "Groovy"

    ```groovy
    localProperties
    ```

=== "Kotlin"

    ```kotlin
    localProperties
    ```
