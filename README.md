## Features

- **Resource Class Generation**

- **Separated Working Space**

- **Manage Dependencies**

- **Work with other Mods**

- **Android Deployment**

- **Download the Game**

- **Asset Management**

- **Debug your Mod**

<br>
<br>

*Supported **Mindustry** versions.*

| Mindustry |    MGPP    |
|:---------:|:----------:|
